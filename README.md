# 🎥 Code-To-Animation

An AI-powered platform that transforms source code into interactive visual animations, making algorithms and program execution easier to understand. The platform leverages Large Language Models (LLMs) to analyze source code and generate step-by-step execution visualizations.

## 🚀 Features

- 🤖 AI-powered code understanding using LLMs
- 🎬 Automatic generation of step-by-step code animations
- 📖 Human-readable explanation of code execution
- 💻 Interactive and responsive web interface
- ☁️ Cloud-deployed full-stack application
- 🔐 Secure backend API integration

## 💡 Motivation

Understanding algorithms and complex code execution through plain text can be difficult for beginners. Code-To-Animation simplifies learning by converting source code into intuitive visual animations, helping students and developers grasp program flow more effectively.

## ⚙️ Workflow

1. User enters source code.
2. Backend sends the code to the LLM.
3. The LLM analyzes the logic and execution flow.
4. Structured animation data is generated.
5. Frontend renders an interactive visualization of the code execution.
